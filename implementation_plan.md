# Phase 1 Implementation Plan - Enterprise Payslip Generator

## Stack
- Plain HTML + Vanilla CSS + JavaScript (no framework, single-file deliverable for Phase 1)
- Google Fonts (Inter)
- Lucide icons via CDN

## Screens to Build
1. Dashboard (Screen A)
2. Global Settings (Screen B) - transport allowance, employer defaults, stored in localStorage
3. Employee Master Data (Screen C) - create/edit/search employees
4. Payslip Input Form (Screen D, steps 1-2 only - no calculation)

## Files
- index.html - main SPA shell
- style.css - design system
- app.js - navigation + form logic + localStorage state
